Ansible Module Reference

Posted on 2019-03-22 | Edited on 2020-01-13 | In Ansible

There are some commonly used Ansible modules that I will reference frequently. More specific details please see Ansible document.

pause

Pauses playbook execution for a set amount of time, or until a prompt is acknowledged. All parameters are optional. The default behavior is to pause with a prompt.
