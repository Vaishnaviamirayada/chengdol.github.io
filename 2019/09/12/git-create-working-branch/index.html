Create Working Branch

Posted on 2019-09-12 | In Git

Let me record the workflow about how to create your own working branch from master.

I git clone the original repo to local, then git checkout to a develop branch based on the master branch, for example:

then go to local git repository

After checkout to new branch, if you get files changed, this is sometimes a format issue, go to edit .gitattributes, comment out and uncomment the pattern with file saved in each action, run git status, then the issue will go.

Now you are in develop branch, let’s check the config information:

Focus on the user.name and user.email, if they are unset, set them by

Then working on develop branch, when run git push origin develop, this will create a pull request and a remote develop branch at first time, then review and merge to master branch on git web.

Because there is no git repos fork, so run git pull origin master in develop branch to get sync up with latest updates from master branch.
